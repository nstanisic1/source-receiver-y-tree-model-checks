Table 26 is the fully quantified benchmark audit-card matrix for the reliability-aware source-receiver model-checking framework. It contains only scoreable focused benchmark cards with terminal counts, country counts, deepest branch counts, source adequacy (`A_s`), diversity contrast (`D`), branch-unit decision states, branch-scale sensitivity summaries, the source-receiver reliability score, and final benchmark decision states.

Rows 1-2 are matched R1a positive comparator cards. They define the source-receiver-compatible comparator envelope used to evaluate the held-out PH908 target application. Row 3 is the held-out PH908 stress test. Rows 4-5 are supporting non-target comparison cards. Rows 6-7 are support-only boundary diagnostics that show why deepest-layer compatibility alone is not sufficient for promotion. The Main Table 1 adds a separate boundary/no-call section for H-M82 and E-M2 without forcing those no-call cards into this numeric schema.

`A_s` is the source terminal count divided by the receiver terminal count for the specified focused comparison. `D` is the natural-log diversity contrast used for the analysis. The six branch-unit decision columns report the deepest, residualized-count, direct-child, depth-2, depth-3, and count-balanced layers. The `nondeep_scoreable_layer_count` and flip-rate fields use the four nondeep sensitivity layers in the Figure 2 highlighted-contract source layer; the residualized-count layer is reported separately as a decision state and is not included in that count. The flip-rate columns and Source-receiver reliability scores are derived from the Figure 2 highlighted-contract source layer and summarize branch-scale sensitivity; The source-receiver reliability score is not a probability, effect size, demographic-disruption measure, or origin score.

The benchmark decision state distinguishes promoted compatible comparators, a held-out rejection, bounded supporting comparisons, and demoted support-only diagnostics. The table is a model-checking and benchmark-control matrix, not an origin table.
