Table 8 provides the row-level source trace for the four named ancient-DNA directness comparisons summarized in Table 10 and shown in the ancient-DNA directness figure. It is not the full plotted ancient-DNA map dataset.

An identified proxy is a published ancient male-line sample retained in the curated pre-550 CE comparison layer with a usable Y-chromosome lineage assignment for the directness audit. A strict direct negative control requires all predefined criteria to be satisfied: distance within 75 km, elevation difference within 250 m, and the same predefined present-day sampling territory. Distance is computed as a Haversine distance from branch-derived PH908 relic-footprint localities to published ancient-sample coordinates.

The `date_median_year_ce` column records the numeric midpoint year used for filtering and distance-layer summaries. Negative values indicate BCE dates in astronomical-style signed year notation.

Failure to satisfy the strict direct negative-control criteria is interpreted only as a boundary on absence testing. It is not positive ancient-DNA evidence for PH908 and is not direct negative evidence for PH908 absence, geographic origin, population continuity, ethnic identity, or migration route.
