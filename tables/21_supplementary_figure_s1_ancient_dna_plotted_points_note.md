Table 21 reports the row-level published ancient-DNA sample points plotted in the Supplementary Figure S1 map panel.

Rows are included after applying the map window used by the figure code (longitude 13.2 to 22.2; latitude 41.3 to 46.5) and the plotted temporal window (2500 BCE to 550 CE, using the median sample date). The table records sample identifiers, source dataset labels, human-readable publication labels, coordinates, date fields, sex, autosomal SNP counts, uniparental markers, the visible-comparator assignment used in the figure, and the coordinate source.

The visible-comparator layer is limited to the E-V13, J2b, R1b, and G2a categories used for the Supplementary Figure S1 coverage display. Two Bubi's cave rows are assigned to the visible-comparator layer by explicit figure-source assignment; the assignment basis is recorded in the table. All rows are plotted for ancient-DNA coverage and visibility context only. They are not evidence of PH908 presence, PH908 absence, geographic origin, ethnic identity, population continuity, or migration route.

Unavailable, unassigned, or not-reported marker fields are retained using the source-layer notation recorded in the plotted data table.
