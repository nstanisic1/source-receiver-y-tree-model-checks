Main Table 1 is a repository source-receiver audit-card matrix.

The table is regenerated from the audited benchmark matrix, branch-unit reliability table, highlighted-contract table, lineage-role table, source-layer table, integrated-boundary summary, and Supplementary Figure S1 completion audit.

Table 1A contains fully quantified audit cards with source and receiver terminal counts, country counts, source adequacy (`A_s`), diversity contrast (`D`), branch-unit status, and final evidence state. These rows are derived from Table 26 and the branch-unit reliability tables.

The perturbation, rarefaction, or sensitivity-status column reports only supported robustness context. For the matched R1a comparators, high-common-n equal-n rarefaction support is reported in Supplementary Table S5 and is treated as an adequacy/power diagnostic, not as a separate origin test.

Table 1B contains boundary/no-call audit cards. H-M82 and E-M2 are included to make abstention behavior visible, but they are not forced into the numeric `A_s,D` comparator schema because diaspora structure, source visibility, or source-receiver independence limits prevent a load-bearing primary decision. Boundary/no-call rows are excluded from numeric comparator summaries unless explicitly scoreable in a separate supplementary analysis.

The table is a model-checking and benchmark-control matrix, not an origin table. It should not be interpreted as evidence of geographic origin, ethnic identity, population continuity, chronology, migration route, ancient-DNA confirmation, absence, or population-level frequency.
