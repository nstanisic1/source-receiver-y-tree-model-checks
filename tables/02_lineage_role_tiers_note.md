# Table 2 Note

Lineage roles indicate how each lineage is used within the reported source-receiver model-checking framework. Main, supporting, and calibration rows are used to evaluate or contextualize predefined source-receiver criteria, whereas boundary, stress, and no-call rows identify settings in which the framework is intentionally not treated as load-bearing evidence. Role assignment should not be interpreted as a direct claim about geographic origin, ethnic identity, population continuity, or migration route.
