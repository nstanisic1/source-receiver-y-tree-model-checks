The layer is defined by three restrictions: each row belongs to the curated PH908 branch-derived locality layer rather than to a modern frequency surface; each locality represents a branch-localized or regionally informative downstream locality rather than an undifferentiated country-level count; and each locality is used only to define where a direct ancient-DNA absence test would require spatial, elevational, and sampling-territory comparability. The layer is not an input to the source-receiver compatibility calculation, the R1a comparator envelope, the adversarial mask search, or the atlas state assignment.
