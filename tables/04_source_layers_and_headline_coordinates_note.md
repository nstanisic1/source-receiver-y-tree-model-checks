# Table 4 Note

Source-layer coordinates summarize predefined metrics calculated from public phylogenetic or project-specific evidence layers. A_s, D, Q2, geographic masks, database snapshots, and inclusion criteria are defined in the Methods. Public-tree and project layers are not treated as independent, population-representative samples. Coordinate values should be interpreted as source-receiver model-check outputs under the specified comparison framework, not as direct claims about geographic origin, ethnic identity, population continuity, migration route, or ancient-DNA confirmation.
