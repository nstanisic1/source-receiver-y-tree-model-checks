# Table 3 Note

Boundary and no-call states define the data-layer and lineage-level conditions under which the source-receiver model-checking framework abstains from load-bearing inference. Q2, D, source-adequacy statistics, rarefaction intervals, permutation tests, and geographic masks are defined in the Methods and applied using predefined criteria. State assignment should not be interpreted as a direct claim about geographic origin, ethnic identity, population continuity, migration route, or absence from currently published ancient-DNA records.
