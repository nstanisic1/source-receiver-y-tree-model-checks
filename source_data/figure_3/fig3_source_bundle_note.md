Figure 3 source bundle note

This source bundle records the input data prepared for Figure 3 before plotting. It separates the primary YFull PH908/R1a source-receiver contrast from supporting branch-resolved layers, bounded direct-child source-adequacy values, branch-unit stability context, and calibration summaries.

The primary plotted values are the YFull coordinates for I-PH908, R-Z280, and R-M458 from Table 4. A_s denotes source adequacy. D denotes ln(q2_receiver/q2_source). The PH908 coordinate is treated as a held-out target application; R-Z280 and R-M458 are matched R1a comparators. The bar panels display A_s and D separately for these three primary YFull rows.

HRAS/YSEQ values are supporting branch-resolved comparisons. FTDNA Discover values are bounded direct-child source-adequacy comparisons; their direct-child D values are not terminal branch-country Q2 contrasts and should not be plotted or interpreted as terminal-Q2 replication.

The branch-unit stability overlay is derived from Tables 11 and 12. It records pass, fail, or no-call status across six evaluated branch-resolution layers for I-PH908, R-Z280, and R-M458. These dots summarize branch-unit stability only; they are not confidence intervals, rarefaction intervals, population-level uncertainty estimates, or independent evidence of historical origin.

The specificity/calibration trace is derived from Table 5. Margin separation, AUROC, permutation support, T95 thresholds, and empirical flag rates are conditional on the predefined PH908-versus-negative-control comparison framework.

Figure 3 source files should not be interpreted as evidence of exact geographic origin, ethnic identity, population continuity, migration route, or ancient-DNA confirmation.
