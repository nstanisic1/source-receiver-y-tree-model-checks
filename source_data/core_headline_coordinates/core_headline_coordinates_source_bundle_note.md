# Core Headline Coordinates Source Bundle

This bundle contains the audited inputs used by `scripts/02_generate_source_layer_headline_coordinates.py` to generate Table 4.

The YFull primary coordinates are read from the clean generated highlighted-contract table, `tables/13_source_receiver_highlight_contracts.csv`. The HRAS/YSEQ rows provide supporting branch-resolved source-adequacy and diversity-contrast coordinates. The FTDNA Discover rows provide bounded direct-child source-adequacy support and a direct-child D value that is explicitly not treated as terminal branch-country Q2 replication.

These files support reproducibility of the source-layer and headline-coordinate table only. They are not independent population-representative samples and should not be interpreted as evidence of exact geographic origin, ethnic identity, population continuity, migration route, or ancient-DNA confirmation.
