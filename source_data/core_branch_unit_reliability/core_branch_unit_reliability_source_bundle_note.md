# Core Branch-Unit Reliability Source Bundle

This bundle contains the audited branch-unit metric source and contract metadata used by `scripts/03_generate_branch_unit_reliability_tables.py` to generate Tables 11 and 12 and the Figure 3 branch-unit stability overlay.

The metric source records the evaluated branch-resolution layers, source and receiver counts, branch counts, source adequacy, diversity contrast, branch-unit state, and supporting sensitivity fields where available. The contract metadata records the descriptive comparison labels, interpretive roles, pattern summaries, and Figure 3 inclusion flags.

Branch-unit reliability states describe whether predefined source-receiver criteria are stable under evaluated branch-resolution layers. They are not confidence intervals, rarefaction intervals, population-representative estimates, or evidence of exact geographic origin, ethnic identity, population continuity, or migration route.
