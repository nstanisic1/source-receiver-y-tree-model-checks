# Core Figure 4 Adversarial Sensitivity Source Bundle

This bundle contains the audited adversarial sensitivity state metrics and descriptive label metadata used by `scripts/05_generate_figure4_adversarial_sensitivity.py` to generate Table 6 and the Figure 4 adversarial sensitivity source files.

The metric source records the ten evaluated PH908 adversarial sensitivity states, matched R1a-comparator envelope thresholds, R-Z280 and R-M458 positive-control values, source-adequacy gaps, diversity-contrast gaps, and required source-adequacy multipliers. The label metadata records the branch-resolution, ambiguity-handling, and source-mask labels used in the generated analysis outputs.

Weak directional compatibility means PH908 source adequacy above one and negative diversity contrast under a tested adversarial state. It is not treated as load-bearing evidence by itself. Matched-envelope entry additionally requires PH908 to enter the state-matched R1a-comparator envelope under the same branch-resolution, ambiguity-handling, and source-mask state.

These files support adversarial sensitivity reproduction only. They are not evidence of PH908 geographic origin, ethnic identity, population continuity, or migration route.
