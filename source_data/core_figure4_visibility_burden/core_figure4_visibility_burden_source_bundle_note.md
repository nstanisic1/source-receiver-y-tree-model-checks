# Core Figure 4 Visibility-Burden Source Bundle

This bundle contains the audited primary visibility-reversal rules used by `scripts/06_generate_figure4_visibility_burden.py`. The generator combines these rules with the repo-contained core YFull highlighted contracts and the Figure 4 adversarial sensitivity metric source to generate the Figure 4 source-visibility burden trace, the Figure 4 primary visibility-bias reversal source file, and the numbered primary reversal table.

The adversarial-state burden trace reports how much PH908 source adequacy would need to increase to reach the state-matched R1a-comparator source-adequacy threshold. The primary reversal summary reports the source-adequacy and source-diversity multipliers required to reach basic compatibility or the less stringent primary matched R1a comparator threshold.

These calculations are sensitivity bounds. They are not corrections for public-tree bias, population-size estimates, geographic-origin claims, ethnic-identity claims, population-continuity claims, or migration-route claims.
